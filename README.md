# Flight Booking System

A console-based flight ticket booking system that allows users to search for flights, manage reservations, and view seat availability.

---

## Features

- Search flights by **departure and arrival airport codes**
- Book flight tickets with:
  - Seat selection
  - Meal options (Normal, Vegetarian, Lactose-free)
- Cancel existing reservations
- View **seat availability map**
- Automatic booking ID generation
- File-based data storage

---

## Technologies

- **C / C++**
- File handling
- Dynamic arrays
- Linked lists

---

## Project Structure

- Flight data stored in `repulojaratok.txt`
- Seat information in `helyek.txt`
- Reservations saved in `foglalasok.txt`
- Seat occupancy tracking in `foglaltsag.txt`
